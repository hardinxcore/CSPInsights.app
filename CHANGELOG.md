# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

## [1.0.0] - 2024-01-30 (Open Source Launch)

### Added
- **Demo Mode**: Integrated sample data generator to test the application without uploading CSVs.
- **Bulk Actions**: Added ability to select multiple customers and update margins or export data in bulk.
- **Snapshot Management**: Added Rename and Delete functionality to the History/Snapshot modal.
- **Legal**: Added MIT `LICENSE` and updated `README.md` with Disclaimer and Privacy Policy.

### Changed
- **UI Improvements**: Refined `Column Chooser` with a grid layout and better visibility.
- **UI Improvements**: Aligned text in `History Modal` using CSS Grid.
- **Branding**: Replaced all "SEVENP" branding with generic "Partner Center" or "Microsoft" assets for open source neutrality.
- **Privacy**: Added explicit privacy notices to the Upload screen.

### Fixed
- Fixed layout alignment issues in the Dashboard filter panel.
- Removed hardcoded proprietary company details from settings defaults.
